1 Reactivity & Binding

Two-way data binding with modifiers, computed properties, and watchers.

sx-text="greeting">

        name: 'Spruce',

        age(val) { if(val > 100) this.age = 100; }

        }">
            <div class="space-y-4">
                <div>

2 Control Flow

Loops with sx-for and memoization with sx-memo. 

<div sx-data="{
            items: ['Apple', 'Banana', 'Cherry'],
            newItem: '',
            add() {
                if(this.newItem) {
                    this.items.push(this.newItem);
                    this.newItem = '';
                }
            },
            remove(idx) { this.items.splice(idx, 1); }
        }">
            <div class="flex gap-2 mb-4">
                <input type="text" sx-model="newItem" sx-on:keydown.enter="add()"
                    placeholder="Add fruit..." class="flex-1 px-4 py-2 rounded-lg border">
                <button sx-on:click="add()" class="bg-blue-600 text-white px-4 py-2 rounded-lg">Add</button>
            </div>
            <ul class="space-y-2" sx-animate>
                <template sx-for="(item, index) in items">
                    <li class="flex justify-between items-center p-3 rounded-lg shadow-sm">
                        <span>
                            <span class="text-slate-400 mr-2">#<span sx-text="index + 1"></span></span>
                            <span sx-text="item"></span>
                        </span>
                        <button sx-on:click="remove(index)" class="text-red-500 text-sm">Remove</button>
                    </li>
                </template>
            </ul>
        </div>

11 Refs

Access DOM elements directly with sx-ref and $refs. 

<div sx-data="{}">
            <input type="text" sx-ref="inputRef">

            <button sx-on:click="$refs.inputRef.focus()">
                Focus Input
            </button>
            <button sx-on:click="$refs.inputRef.value = 'Hello!'">
                Set Value
            </button>
        </div>

12 Error Fallback

Gracefully handle missing data or errors with sx-error-fallback.

Attempting to read broken.value:

Without fallback, this would log an error to the console. 

<div sx-data="{ broken: null }">
            <p sx-text="broken.value"
               sx-error-fallback="Error: Data is missing">
            </p>
        </div>

# CHANGELOG

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.4.0] - 2025-11-27

### Added

- sx-animate for smooth list animations (add/remove/move)
- Memory leak fixes for sx-for and store subscribers
- Function caching for expressions
- Batched updates via microtask
- Deep reactivity for nested objects
- sx-class preserves original classes
- Network requests check response.ok
- Memo deps work with loop locals

## [0.1.0] - 2025-11-27

### Added

- sx-data
- sx-class
- sx-for
- sx-show
- sx-memo
- sx-store
- sx-expr
